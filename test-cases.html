TC-001: Dashboard Page Load

Objective: Verify dashboard loads correctly with all components.

Steps:

  1. Navigate to dashboard.html
  2. Verify page loads without errors
  3. Check all sidebar navigation items display
  4. Verify header with title and search bar appear

Expected Result: Page loads successfully with all UI elements visible and responsive.

TC-002: Stat Cards Display

Objective: Verify all stat cards display correct information with icons.

Steps:

  1. Load dashboard page
  2. Verify 4 stat cards are visible (Projects, Tasks, Team, Completion Rate)
  3. Check each card displays correct number values
  4. Verify icons appear with 30% opacity in top-right corner
  5. Verify sub-text shows relevant metrics

Expected Result: All stat cards display correctly with accurate values and icons.

Pass
